# Bubble sort

## Description
This is an algorithm to implement the bubble sort in ruby

## Built With
  - Ruby
  - Rubocop
  
## Usage
  - git clone https://github.com/oloomoses/bubble-sort.git
  - cd bubble-sort
  - git checkout development
  - ruby bubble_sort.rb
